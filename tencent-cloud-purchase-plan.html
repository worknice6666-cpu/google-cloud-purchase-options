用于承载中国大陆为主的小程序、国内 App、微信生态工具和国内用户访问入口。建议按一年服务额度采购，避免每个小程序或 App 单独重复走公司预算申请。

一、总体建议

腾讯云建议作为中国大陆产品的补充云平台，和 Google Cloud 形成双云分工：Google Cloud 继续服务全球团队与海外系统，腾讯云承载国内小程序、国内 App、微信登录/支付、国内 CDN、备案、短信和合规访问入口。

30,000 建议先采购 RMB 30,000 / 年。适合国内小程序、1-2 个 App/工具和基础生产环境。
CN 国内用户访问、备案、微信生态、短信、CDN 和内容合规能力放在腾讯云。
60k+ 如果多个国内产品快速上线，可扩展到 RMB 60,000-100,000 / 年。

二、如何采购

确认年度预算

内部申请腾讯云年度服务额度 RMB 30,000 / 年，用于国内小程序、国内 App 和微信生态产品优先上线。

企业实名认证与合同发票

使用公司主体完成腾讯云企业认证，确认合同、发票、付款方式和采购流程。若公司要求供应商合同，可联系腾讯云销售或代理商。

准备域名与备案

国内小程序和 App 建议提前准备国内域名、ICP备案、App 备案、小程序备案所需资料，避免开发完成后卡在审核环节。

先开通基础服务

优先开通 DNSPod、CVM/轻量应用服务器、CloudBase、TencentDB、COS、CDN、SSL、CLS、云监控、CAM、短信等基础服务。

设置预算告警

设置年度预算、月度预算、80% 告警、100% 告警和超额提醒，尤其关注 CDN、短信、日志、对象存储和数据库费用。

三、采购服务内容

以下服务建议纳入腾讯云年度采购范围，用于国内正式域名、备案、小程序后端、国内 App API、文件存储、短信、日志、安全和后续扩展。

服务 是否购买 用途说明
CloudBase 云开发 建议购买 适合微信小程序、Web、移动 App、管理后台和 AI 应用的 Serverless 后端，包含数据库、存储、身份认证、云函数、云托管等能力。
CVM / 轻量应用服务器 必买 承载国内 API、管理后台、定时任务、反向代理或传统后端服务。轻量应用服务器适合起步，CVM 适合正式生产扩展。
TencentDB MySQL / PostgreSQL 必买 存储用户、订单、内容、配置、项目数据和业务记录。小程序轻量场景也可先用 CloudBase 数据库。
COS 对象存储 必买 存放图片、附件、导出文件、报表、用户上传素材、备份文件，并可与 CDN 配合加速。
DNSPod + SSL 证书 必买 管理国内域名解析、正式 HTTPS 域名和小程序合法域名。
CDN / EdgeOne 建议购买 加速国内图片、静态资源、下载文件和 H5 页面；访问量增长后可升级安全与边缘能力。
CLB + WAF 按需购买 正式公网入口、负载均衡和 Web 安全防护。早期轻量项目可先不开 WAF，公网业务稳定后补上。
短信 SMS 建议购买 国内手机号登录、验证码、通知短信。短信量可能增长较快，建议单独设置额度。
CLS 日志服务 + 云监控 必买 日志、告警、错误排查、访问监控、费用观察和稳定性分析。
CAM + KMS / 凭据管理 必买 团队权限、子账号、服务账号、API Key、数据库密码和第三方密钥管理。
内容安全 按需购买 如有用户生成内容、评论、图片上传、AI 生成内容，建议接入文本/图片内容安全审核。
年度采购金额建议：RMB 30,000 / 年 这笔费用适合国内小程序/国内 App 的第一阶段上线，覆盖基础服务器、数据库、对象存储、CDN、短信、日志、备案相关资源和基础安全能力。AI 模型 API、内容审核高调用量、短信高调用量建议单独设置上限。

四、适用产品范围

优先适合腾讯云承载

  • 微信小程序、公众号 H5、企业微信工具。
  • 国内 App 的 API 入口、短信、文件、图片和下载资源。
  • 国内用户访问频率高的工具类产品。
  • 涉及微信登录、微信支付、OpenID/UnionID 的产品。
  • 需要 ICP 备案、App 备案、小程序备案的国内业务。

不建议放在腾讯云的部分

  • 海外团队内部工具仍可继续放 Google Cloud。
  • 全球统一后台和海外数据分析仍可放 Google Cloud。
  • 外部 AI 模型调用费用不建议混入腾讯云基础云预算。
  • 强依赖海外第三方 API 的服务需评估国内访问稳定性。

六、付费方式与采购注意事项

方式 建议
包年包月 适合 CVM、数据库、固定带宽、部分安全服务等长期稳定资源，建议按一年购买。
资源包 / 预付费包 适合 COS、CDN、短信、日志等可预估消耗的服务，便于控制年度预算。
按量计费 适合早期测试或不确定用量的服务，但需要配预算告警，避免突发费用。
腾讯云销售 / 代理商合同 如果公司需要合同、发票、年度采购流程，建议联系腾讯云销售或授权代理商统一报价。
不建议 不建议用个人账号、个人付款方式或每个小程序单独采购生产资源。

七、与 Google Cloud 的分工

平台 定位
Google Cloud 全球团队工具、海外官网、海外 AI 工具、全球后台、海外数据服务。
腾讯云 中国大陆小程序、国内 App 入口、国内 CDN、微信生态、备案、短信、国内用户访问。
最终采购口径 建议今年补充采购 RMB 30,000 / 年的腾讯云国内产品服务额度，作为 Google Cloud 全球平台的国内补充，用于承载小程序、国内 App 和微信生态相关服务。
